Opdracht 3

Waarom staat de tekst in de blokken horizontaal gecentreerd?

Omdat er op het element een 'text-align: center;' werd toegepast.

Wat is het veschil tussen vertical-align en text-align

text-align= een CSS-eigenschap, die wordt gebruikt om de tekst uit te lijnen binnen een alinea, div, ... (zoals centreren, links, rechts.)

vertical-align= bepaalt hoe elementen die naast elkaar op een lijn zijn geplaatst, worden uitgelijnd.

Wat is de ‘simpele’ manier om een één enkele regel tekst verticaal te centreren?

Voeg dit toe in de css-code op de klasse die je wil centreren:

.center {

line-height: 200px;

height: 200px;

text-align: center;

}

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren?

Omdat de eigenschap vertical-align werkt met tabelcellen, kan je de bovenliggende div instellen als een CSS-tabel en kan je de onderliggende div instellen als een tabelcel, gevolgd door het gebruik van vertical-align: middle om de inhoud van de onderliggende div verticaal te centreren. Deze methode werkt met meerdere regels tekst en de container-div groeit dynamisch mee met de inhoud.

In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang?

Omdat de regel .tall meer op een lagere lijn staat dan .area. Hoe verder in de code iets staat, hoe meer prioriteit eraan wordt gegeven.

Opdracht Lorem Ipsum

Maak een HTML5 pagina met 2 lorem ipsum sections van voldoende lengte.

Waarom staan de kolommen niet naast elkaar, maar onder elkaar?

De kolommen staan onder elkaar omdat de browser een automatische gutter voorziet bij deze toepassing, waardoor er dus geen plaats is voor de gutter bij de opstetlling 50% voor kolom één en 50% voor kolom twee. De tweede kolom wordt automatisch naar de volgende 'regel' geschoven.

Bij een kolomwidth van 47% is er wel ruimte voor deze gutter en zullen de kolommen naast elkaar weergegeven zijn.
