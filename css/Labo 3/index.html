Opdracht 3

Waarom heeft de CSS regel
.onbekend {background‐color : red; }
geen effect als je deze toepast op een <td> cel in een <tr> rij met class="alt", maar wel in een <tr> rij zonder deze 'alt' class?

De rode kleur komt niet boven de blauwe rijkleur omdat de rij een hogere prioriteit heeft dan het <td> element.

Dit kan aangepast worden door de regel
'!important'
toe te voegen aan de CSS-regel waar de rode kleur wordt toegevoegd. Soms kan de volgorde van CSS regels ook invloed hebben op de prioriteit, maar in deze situatie is dat niet het geval aangezien de rode kleur helemaal onderaan in het CSS-bestand toegewezen wordt.
